Design and build a formal, modern, and easy-to-use school website for schools under the same organization.

Organization details:
- Organization name: Michigan Tamil Sangam (MTS)
- Logo file: `organization-logo.png`

Primary goal:
- Provide a clean login experience and a professional home page for school users.

Target users:
- Students, teachers, and school administrators within MTS schools.

Core pages and flow:
1. Login Page
- Include fields for username and password.
- Include a clear `Sign In` button.
- Show friendly validation messages for empty or invalid inputs.
- On successful login, redirect the user to the Home page.

2. Home Page
- Show a top navigation bar across the page.
- Display school name prominently in the header area.
- Show user profile name and profile photo at the top-right corner.
- Use a simple layout that allows quick access to main sections.

Header and navigation requirements:
- Top menu bar should remain clean and easy to read.
- Include organization branding (MTS name and logo).
- Keep navigation labels short and clear.
- Ensure menu works well on mobile (collapse into a hamburger menu on small screens).

UI/UX expectations:
- Formal school website style: professional, trustworthy, and organized.
- Keep the interface simple and beginner-friendly.
- Prioritize readability with clear typography and spacing.
- Use consistent colors, buttons, and form styles across pages.
- Avoid clutter; focus on essential actions.

Responsive design requirements:
- Fully responsive on desktop, tablet, and mobile.
- Use fluid layouts and flexible spacing.
- Ensure text, buttons, and form fields are easy to tap on touch devices.
- Test key breakpoints (mobile, tablet, laptop, wide desktop).

Accessibility and quality:
- Use semantic HTML structure.
- Ensure sufficient color contrast for text and controls.
- Provide visible focus states for keyboard navigation.
- Add alt text for logo and profile image.

Output expectation:
- Deliver a clean, responsive school website UI with login and home pages, aligned to MTS branding and formal school standards.
